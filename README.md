# Sistema de Gerenciamento de Estacionamento

Projeto desenvolvido em Java como trabalho acadêmico.

## Funcionalidades
- Entrada e saída de veículos (Carro, Moto, Van)
- Cálculo automático de tarifas por tempo
- Consulta de vagas
- Visualização de ocupação do estacionamento
- Relatório financeiro
- Salvamento e carregamento de dados em arquivo

## Tecnologias utilizadas
- Java
- Manipulação de arquivos (FileWriter, Scanner)
- Estruturas de dados (ArrayList, matrizes)

## Como executar
1. Compile:
   javac trabalhoteste.java

2. Execute:
   java trabalhoteste

## Aprendizados
- Lógica de programação aplicada
- Validação de entrada de dados
- Estruturação de sistema baseado em menus
- Manipulação de arquivos em Java

## Melhorias futuras
- Refatoração para Programação Orientada a Objetos (POO)
- Interface gráfica
- Integração com banco de dados
